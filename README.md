## About StudyPack
StudyPack은 개인 과외나 학원 수업 과정에서의 편의를 위해 작성되었습니다. <br>
개인이 작성한 통합된 예제나 모의고사, 기출문제와 같은 데이터들을 데이터베이스에 분리해서 저장하고 <br>
개별 이용자가 필요로 할 때, 원하는 내용만 모아 하나의 문서로 파일을 제공하는 서비스입니다 <br>

django framework 를 기반으로 작동하도록 개발되었습니다. <br>
HTML, CSS, JS를 사용해 사용자에게 보여지는 화면을 구성하였습니다. <br>

## Features

- 사용자가 여러 문제가 담긴 파일을 서버에 제출하면 서버에서 문제들을 하나씩 분리해서 저장함 <br>
- 사용자가 특정 유형, 조건 등으로 문제들을 요청하면 서버에서 해당하는 문제들을 엮어 하나의 문서 파일로 제공함 <br>
- 현재 저장된 파일들의 목록을 제공함 <br>
