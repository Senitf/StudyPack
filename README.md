## About StudyPack
StudyPack은 개인 과외나 학원 수업 과정에서의 편의를 위해 작성되었습니다. <br>
개인이 작성한 통합된 예제나 모의고사, 기출문제와 같은 데이터들을 데이터베이스에 분리해서 저장하고 <br>
개별 이용자가 필요로 할 때, 원하는 내용만 모아 하나의 문서로 파일을 제공하는 서비스입니다 <br>

## Features

- 개별 음악 업로드와 동시에 유튜브 영상 제공
- 로그인된 사용자에 한하여 여러 이용자들이 업로드한 음악들을 조합하여 플레이리스트 생성 가능
- 개별 음악과 플레이리스트에 대한 좋아요, 저장 기능
